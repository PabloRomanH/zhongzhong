Keyboard shortcuts

In order to make Zhongzhong work in input fields and text areas, hold down the Alt-key on your keyboard.

You can disable the keyboard shortcuts for quick settings if you don't use them and keep hitting the keys by mistake.

You can disable the keyboard shortcuts to look up online resources if you don't use them and keep hitting the keys by mistake.

Keyboard actions:

Alt +  : Enable/Disable Zhongzhong
  
N : Next word
B : Previous character
M : Next character
  
V : Read word aloud
Shift/Enter : Switch dictionaries
  
A : Alternate popup location
Y : Move popup location down
X : Move popup location up
  
C : Copy to clipboard
  
S : Add word to Skritter queue
  
R : Save word to the internal word list
Alt + W : Show the word list

Quick settings:

D : Display/hide definitions
P : Display/hide Pinyin
Z : Display/hide Zhuyin (Bopomofo)

Look up selected text in online resource:

Alt + 1 : LINE Dict (formerly nciku)
Alt + 2 : YellowBridge
Alt + 3 : Dict.cn
Alt + 4 : iCIBA
Alt + 5 : MDBG
Alt + 6 : JuKuu (manual search)
Alt + 7 : Moedict
Alt + 8 : Baidu Baike
T : Tatoeba
K : Study character in Koohii.com (NEW)
